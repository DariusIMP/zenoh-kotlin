fun scout(callback: Callback<Hello>, whatAmI: Set<WhatAmI> = setOf(Peer, Router), config: Config? = null): Result<Scout<Unit>>

Scout for routers and/or peers.

Scout spawns a task that periodically sends scout messages and waits for Hello replies. Drop the returned Scout to stop the scouting task or explicitly call Scout.stop or Scout.close.

Return

A result with the Scout object.

Parameters

callback

Callback to be run when receiving a Hello message.

whatAmI

WhatAmI configuration: it indicates the role of the zenoh node sending the HELLO message.

config

Optional Config for the scout.
